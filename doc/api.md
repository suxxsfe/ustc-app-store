## GET /api/appinfo  

params  

| 名称 | 解释 | 示例 |
| ---- | ---- | ---- |
| \_id | app id | `""` |  

returns  

| 名称 | 解释 | 示例 |
| ---- | ---- | ---- |
| name | app 名 | `"app name"` |
| authors | 作者列表 | `[user]` |
| createdate | 创建日期 | `"2023-11-11"` |
| updatedate | 修改日子 | `"2023-11-11"` |
| links | 相关链接 | `[{webname: "link name", url: "www.xxx.yyy"}]` |
| tags | 标签 | `[{name: "tag name", _id: 123}]` |
| platform | 运行平台 | `["Windows", "Web"]` |
